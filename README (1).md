# 統合管理システム Pro

廃棄物収集運搬業向けの統合管理システム

## 機能

### ✅ 実装済み
- 👥 顧客管理
  - 顧客情報の登録・編集・削除
  - 契約書管理（複数PDF対応）
  - 回収時の注意事項管理
  - 画像添付機能
- 📦 コンテナレンタル管理
  - 7種類のコンテナタイプ対応
  - 請求日管理・通知機能
  - レンタル状況の可視化

### 🔮 開発予定
- 🚗 車両管理
- 📊 配送実績管理
- 💰 原価計算
- 📈 分析・レポート

## 使い方

1. `index.html`を開く
2. GAS_API_URLを自分のURLに変更
3. ブラウザで表示

## 技術スタック

- HTML5 / CSS3 / JavaScript
- Google Apps Script
- Google Sheets
